    C++ iterator

    c++ iterator

    Date : 2019/11/18

    Ref : eehoeskrap 블로그 tcpschool

    반복자란?

    STL 컨테이너에 저장된 요소를 반복적으로 순회하여, 각각의 요소에 대한 접근을 제공하는 객체다.

    반복자의 종류

    반복자 종류 사용 방식
    입력 반복자 (input iterator) istream_iterator
    출력 반복자 (output iterator) ostream_iterator inserter front_inserter back_inserter
    순방향 반복자 (forward iterator)
    양방향 반복자 (bidirectional iterator) list set 과 multiset map 과 multimap
    임의접근 반복자 (random access iterator) 일반 포인터 vector deque

    반복자의 역할

    반복자 종류 읽기 접근 쓰기 증감 비교
    입력 반복자 (input iterator) =*p -> ++ == !=
    출력 반복자 (output iterator) *p= ++
    순방향 반복자 (forward iterator) =*p -> *p= ++ == !=
    양방향 반복자 (bidirectional iterator) =*p -> *p= ++ -- == !=
    임의접근 반복자 (random access iterator) =*p -> [] *p= ++ -- + - += -= == != < > <= >=
