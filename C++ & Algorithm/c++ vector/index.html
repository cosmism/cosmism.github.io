    C++ vector

    c++ vector

    Date : 2019/11/18

    Ref : blockdmask 블로그

    header

    #include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

    후에 벡터와 정렬등의 알고리즘을 사용할 예정이어서 vector와 algorithm헤더를 가져와준다.

    vector init

    vector<int> v;
// 비어있는 int 형 vector v를 선언
vector<int> v(5);
// 0으로 초기화 된 5개의 원소를 가지는 int 형 vector v를 선언
vector<int> v(5, 2);
// 2로 초기화 된 5개의 원소를 가지는 int 형 vector v를 선언

    이후 출력해보면 다음과 같다.

    vector<int> vec1(5);
for (auto it : vec1){
    cout << it << " ";
}
cout << endl;
# 0 0 0 0 0
vector<int> vec2(5, 2);
for (auto it : vec2){
    cout << it << " ";
}
cout << endl;
# 2 2 2 2 2
vector<int> vec3(vec2);
for (auto it : vec3){
    cout << it << " ";
}
# 2 2 2 2 2

    vector function

    assign : 원소 할당(초기화)

    vector<int> vec(3,1);
for (auto it : vec){
    cout << it << " ";
}
cout << endl;
# 1 1 1
vec.assign(5, 2);
for (auto it : vec){
    cout << it << " ";
}
# 2 2 2 2 2

    at : 원소 참조

    vector<int> vec;
vec.push_back(1);
vec.push_back(2);
vec.push_back(3);
cout << vec.at(2) << endl;
# 3

    범위를 점검해서 안전하고 느리다.

    cout << vec[2] << endl;
# 3

    위험하고 빠르다.

    front() : 가장 앞의 원소를 가져온다.

    end() : 가장 뒤의 원소를 가져온다.

    vector<int> vec;
vec.push_back(1);
vec.push_back(2);
vec.push_back(3);
cout << vec.front() << " " << vec.back() << endl;
# 1 3

    push_back() : 원소를 뒤에서 삽입한다.

    pop_back() : 원소를 뒤에서 제거한다. 

    vector<int> vec;
vec.push_back(1);
vec.push_back(2);
vec.push_back(3);
vec.push_back(7);
cout << vec.back() << " ";
# 7
vec.pop_back();
cout << vec.back() << " ";
# 3

    vec.begin() : 첫 번째 원소를 가리킨다. ( iterator )

    vec.end() : 마지막의 다음 원소를 가리킨다. ( iterator )

    algorithm의 sort를 이용한 정렬

    vector<int> vec;
vec.push_back(3);   
vec.push_back(2);
vec.push_back(5);
vec.push_back(1);
sort(vec.begin(), vec.end());
for (auto it : vec){
    cout << it << " ";
}

    내림차순 정렬

    #include <functional>

int main(){
    vector<int> vec;
    vec.push_back(3);
    vec.push_back(2);
    vec.push_back(5);
    vec.push_back(1);
    sort(vec.begin(), vec.end(), greater<int>());
    for (auto it : vec){
        cout << it << " ";
    }
}
# 5 3 2 1

    greater사용을 위해 functional헤더를 가져온다.
