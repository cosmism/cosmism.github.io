    Mnist with mlp, cnn and rnn

    Date : 2019/11/17

    Ref : 케라스로 구현하는 고급 딥러닝 알고리즘 [chapter 1]

    dense레이어와 conv, simpleRnn레이어에서 요구하는 input shape이 각각 다르기 때문에, 레이어에 맞게 데이터의 shape을 변경해주어야 한다.

    mnist의 이미지 크기는 28*28이다.

    mlp input shape

    28 * 28 크기의 60000장 이미지[ (60000, 28, 28) ]를 784(이미지의 넓이) * 60000 장 형태[ (60000, 784) ]로 만들어야 한다.

    x_train = np.reshape(x_train, [-1, input_size])
x_test = np.reshape(x_test, [-1, input_size])
input_size = image_size * image_size
model.add(Dense(hidden_units, input_dim=input_size))

    cnn input shape

    cnn은 input_shape=(n, m, 3) 의 형태가 되어야 한다. 마지막 인자는 rgb(3)인지, grayscale(1)인지를 뜻한다. 

    x_train = np.reshape(x_train,[-1, image_size, image_size, 1])
x_test = np.reshape(x_test,[-1, image_size, image_size, 1])
input_shape = (image_size, image_size, 1)
model.add(Conv2D(filters=filters, kernel_size=kernel_size, activation='relu', input_shape=input_shape))

    rnn input shape

    rnn은 책의 텍스트가 매끄럽지 않고, 설명도 부족해서 블로그를 참고했다.

    rnn은 (batch_size, timesteps, input_dim)의 형태가 되어야 한다.

    • batch_size : 데이터 인스턴스의 갯수
    • timesteps : 인풋 시퀀스의 갯수 # input_length
    • input_dim : 각 인풋 시퀀스의 차원 # 입력의 크기 
    X_train = np.reshape(x_train,[-1, image_size, image_size])
input_shape = (image_size, image_size)
model.add(SimpleRNN(units=units, dropout=dropout, input_shape=input_shape))
# input shape (timesteps(28), input_dim(28))
    • return_sequences : 출력 시퀀스에서 마지막 출력을 반환할지 또는 전체 시퀀스를 반환할지 여부
    from keras.layers import Input
inputs = Input(shape=(28,28))
rnn = SimpleRNN(50, return_sequences = True)(inputs)
# rnn.shape (?, ?, 50)
rnn = SimpleRNN(50, return_sequences = False)(inputs)
# rnn.shape (?, 50)
