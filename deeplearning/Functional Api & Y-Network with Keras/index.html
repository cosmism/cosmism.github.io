    Functional Api & Y Network with Keras

    Functional Api & Y-Network with Keras

    Date : 2019/11/17

    Ref : 케라스로 구현하는 고급 딥러닝 알고리즘 [chapter 2]

    functional api

    input레이어를 따로 가져와준다.

    from keras.layers import Input

    순차형과는 코드가 조금씩 다르다. 후에 확장성을 고려하여 함수형으로 케라스 모델을 만드는 연습을 하자.

    inputs = Input(shape=input_shape)
y = Conv2D(filters=filters,
           kernel_size=kernel_size,
           activation='relu')(inputs)
y = MaxPooling2D()(y)
y = Conv2D(filters=filters,
           kernel_size=kernel_size,
           activation='relu')(y)
y = MaxPooling2D()(y)
y = Conv2D(filters=filters,
           kernel_size=kernel_size,
           activation='relu')(y)
y = Flatten()(y)
y = Dropout(dropout)(y)
outputs = Dense(num_labels, activation='softmax')(y)

model = Model(inputs=inputs, outputs=outputs)
model.summary()

    Y-Network

    y-network를 활용하면 mlp모델과 cnn모델을 합쳐서 사용할 수 있다. 순차형보다 함수형 api가 편한 이유 중 하나다.

    사용하는 방법도 생각보다 간단하다.

    for i in range(3):
    x = Conv2D(filters=filters,
               kernel_size=kernel_size,
               padding='same',
               activation='relu')(x)
    x = Dropout(dropout)(x)
    x = MaxPooling2D()(x)
    filters *= 2

    모델 y도 비슷하게 이런식으로 만들어준다.

    이후 두 모델을 합치고, flatten, model을 만든다.

    y = concatenate([x, y])
y = Flatten()(y)
y = Dropout(dropout)(y)
outputs = Dense(num_labels, activation='softmax')(y)
model = Model([left_inputs, right_inputs], outputs)
plot_model(model, to_file='cnn-y-network.png', show_shapes=True)

    이후에는 순차형 api와 같이 comple, fit 등등의 함수를 사용하면 된다.
